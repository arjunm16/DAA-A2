Line Fitting Dynamic Programming Algorithm (Segmented Least Squares)

  • Segmented least squares is a method used to fit a piecewise linear function to a set of data points. It divides the data into segments and fits a linear regression line to each segment.

  • Dynamic programming can be used to efficiently solve the problem of finding the optimal segmentation of the data, minimizing the sum of squared errors of the linear fits. The algorithm works as follows:

    1. Initialize two arrays, S and B, each of size n+1, where S[j] is the minimum sum of squared errors for segmenting the first j data points, and B[j] is the index of the first data point in the last segment of the optimal segmentation of the first j points.
    2. Initialize S[0] to 0.
    3. For j from 1 to n:
      1. Set S[j] to infinity.
      2. For i from 1 to j:
        1. Compute the sum of squared errors E(i,j) for the segment from i to j.
        2. Set cost = S[i-1] + E(i,j).
        3. If cost < S[j], update S[j] to cost and B[j] to i.
    4. Backtrack from B[n] to construct the optimal segmentation.

  • The time complexity of this algorithm is O(n^3) where n is the total number of points.

  • It is important to note that this worst-case time complexity is not always reached in practice, and the actual running time may be much faster depending on the specific structure of the graph and the implementation of the algorithm.
